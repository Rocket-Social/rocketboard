# Support

## Getting Help

Use these channels for support:

- Read [README.md](./README.md) for setup and repository scope.
- Read [SELF_HOSTING.md](./SELF_HOSTING.md) for deployment and self-hosting guidance.
- Open a GitHub issue for reproducible bugs, documentation gaps, or feature requests.

## Before You Open an Issue

Please include:

- Rocketboard version or commit SHA
- whether you are running the managed service mirror or a self-hosted deployment
- Node.js version
- Supabase CLI version, if relevant
- exact reproduction steps
- logs, stack traces, or screenshots when available

## Security Issues

Do not use public issues for security reports. Follow [SECURITY.md](./SECURITY.md) instead.
